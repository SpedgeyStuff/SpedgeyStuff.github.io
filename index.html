Mock E-Commerce Website

This website was made for my E-Commerce module, and was my first time building a (almost) complete website. It was made using HTML, CSS, and a local Apache server with a MySQL database to handle the product data. I used bootstrap to ensure it was mobile-friendly without having to write a load of media queries.


3D OpenGL Game

This game was made as a part of my Advanced Games Technology module, where, starting with a basic template, we were required to implement a number of features and objects into the game. This included a unique camera technique (third person in my case), basic state-machine based AI, physics, collision detection and response, OpenGL primitive-based meshes as well as external meshes with proper textures and normals.

2D Java Game

Reincarnation

This work was produced as a part of my A-Level graphics course, where the theme I chose (from those given as a part of the exam) was reincarnation. I used this theme to explore the aesthetics surrounding different cultural interpretations of death and reincarnation. These were Mexico's Dias De Muertos, Ghanaian fantasy coffins, and paisley patterns along with Hinduism.
